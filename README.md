## 🚀 Key Features

- **Chronological Timeline**: Visualize your progress through a sleek, interactive vertical journey.
- **Apex Analytics**: Categorized distribution of "Work", "Life", and "Vitality" breakthroughs.
- **Premium Design System**: "Emerald Matrix" aesthetic with glassmorphism, dynamic gradients, and professional typography.
- **Standardized API**: Robust communication layer with specialized response helpers and error handling.
- **Full Responsiveness**: Seamless experience across mobile, tablet, and desktop devices.
- **Security**: JWT-based authentication and secure password hashing.

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Notifications**: React Hot Toast
- **State/Routing**: React Context API & React Router

### Backend
- **Server**: Node.js & Express
- **Database**: MongoDB (Mongoose)
- **Security**: JWT, bcryptjs
- **Environment**: Dotenv

## 📦 Installation & Setup

### Prerequisites
- Node.js (v14+)
- MongoDB (Local or Atlas)

### 1. Clone the Repository
```bash
git clone https://github.com/mrankitkumar/Personal-Milestone-Assignment.git
cd Personal-Milestone-Assignment
```

### 2. Backend Setup
```bash
cd backend
npm install
```
Create a `.env` file in the `backend` folder:
```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```
Start the server:
```bash
npm run dev
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
```
Start the development server:
```bash
npm run dev
```
